## Folder Structure
```md
./
└─ [category]/
    └─ [challenge-title]/
        ├─ public/
        │   └─ (files that are given to participants)
        ├─ src/
        │   └─ (source if there's any)
        ├─ writeup/
        │   └─ (solution for the problem, if there's any)
        ├─ docker-compose.yml (if there's any)
        └─ README.md
```
