## Folder Structure
```md
./
└─ [category]/
    └─ [challenge-title]/
        ├─ public/
        │   └─ (files that are given to participants)
        ├─ src/
        │   └─ (source for the problem service or to generate attachment)
        ├─ writeup/
        │   └─ (solution for the problem)
        ├─ docker-compose.yml (for problem services)
        └─ README.md
```
